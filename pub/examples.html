Static Demo

An example of a static music staff generated by note.js.

See Example
  • HTML
  • Javascript

After adding the library to your public folder, add the following to your code:

<link rel="stylesheet" type="text/css" href="noteJSstyles.css">

<script defer type="text/javascript" src='noteJS.js'></script>

const staff = new Staff('target', '4/4', 'treble', 'G', '145', odeToJoy, 'default');

staff.build(staff);

Note: odeToJoy is a composition. Look here for instructions on how to properly create a composition of your own.

Background Music Demo

If you want complete control, you can hide both the staff and the music playback controls.

See Example
  • HTML
  • Javascript

After adding the library to your public folder, add the following to your code:

<link rel="stylesheet" type="text/css" href="noteJSstyles.css">

<script defer type="text/javascript" src='noteJS.js'></script>

const staff = new Staff('target', '4/4', 'treble', 'G', '145', odeToJoy, 'hidden');

staff.build(staff);

staff.playbackControl('play');

Note: odeToJoy is a composition. Look here for instructions on how to properly create a composition of your own.
