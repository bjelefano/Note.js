Getting Started


First, download the library from here.

Extract all of the files to your public directory. Then, add the following to your HTML header:.

<link rel="stylesheet" type="text/css" href="noteJSstyles.css">

<script defer type="text/javascript" src='noteJS.js'></script>


Staff Object Parameters


ParameterTypeDescription
containerStringThe HTML element where the staff will be built.

Either set to "body" or the ID of the target element.
timeSignatureStringA string in the form '[0-9]/[0-9]'.

Indicates the Time Signature of the staff.

This paramater does not affect functionality; its purely visuals.
clefStringEither 'treble' or 'bass'.

Indicates the desired clef you want displayed on the staff.

This affects the tone of certain notes.
keyStringA string in the form '[A-Ga-g][#b]?'.

Indicates the key signature of the staff.

This affects the tone of certain notes.
tempoIntA positive integer.

Indicates the tempo of the staff.

This affects speed at which playback occurs.
compositionString Indicates which notes are rendered on the staff element when the build() method is called.

See below for format.
modeStringThis should be set to one of the following:
  • 'default'
  • 'editable'
  • 'only-controls'
  • 'hidden'
Indicates how the staff/controls are displayed. See the example page to see what each setting does.

Staff Object Methods


MethodParametersDescription
buildNoneBuilds the staff element.
editWaveTypewaveType:StringChanges the wavetype used by the MIDI controller.

The parameter 'waveType' should be set to one of the following:
  • 'sawtooth'
  • 'square'
  • 'sine'
  • 'triangle'
changeModemode:StringChanges how the staff/controls are displayed.

See the description for the object parameter "mode" for appropriate inputs.
setCompositioncomposition:StringRerenders the staff based on a new set of notes (determined by the input).

See below for the proper format of a composition.
playbackControlsetting:StringPlays/Pauses/Stops the music.

Here are the only valid inputs:
  • 'play'
  • 'pause'
  • 'stop'

Composition Format


Text in brackets indicate options.

Element TypeFormatExample
Note{(quarterNote|halfNote|wholeNote) (line|space)[0-5]}{quarterNote line3}
Rest{(quarterRest|halfRest|wholeRest)}{halfRest}
Accidental{(sharpAccidental|flatAccidental|naturalAccidental) (line|space)[0-5]}{naturalAccidental space1}
Bar{Bar}{Bar}

Elements in the composition are seperated by dashes ('-').


Here is a sample composition:

"{quarterNote line3}-{naturalAccidental space1}-{wholeNote space1}-{halfRest}-{bar}"
